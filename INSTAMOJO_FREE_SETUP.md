## Step 2: Complete Basic Profile (3 minutes)

1. Login to dashboard: https://www.instamojo.com/dashboard/
2. Go to Settings → Account
3. Fill basic business details:
   - Business name
   - Category
   - Website (https://aatra-ai.vercel.app)
4. Save

## Step 3: Get API Credentials (2 minutes)

1. Dashboard → Settings → API & Webhooks
2. You'll see:
   - **Client ID** (Public key)
   - **Client Secret** (Private key)
3. Copy and save both keys

## Step 4: Add Keys to Vercel (5 minutes)

1. Go to Vercel: https://vercel.com/dashboard
2. Select Aatra.ai project
3. Settings → Environment Variables
4. Add these variables:
   ```
   VITE_INSTAMOJO_CLIENT_ID=your_client_id_here
   VITE_INSTAMOJO_CLIENT_SECRET=your_secret_key_here
   VITE_ADMIN_EMAIL=santhoshkrishna958@gmail.com
   ```
5. Redeploy (automatic)
